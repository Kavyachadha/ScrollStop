# ScrollStop 🚫📱

ScrollStop is a Windows-based screen time tracker that helps you control excessive scrolling on apps like Instagram, YouTube, Reddit, etc.

## 🚀 Features
- Tracks active window usage
- Daily time limits per app
- Overuse alerts with snooze option
- Streak tracking system
- Web dashboard (Flask)

## 🛠️ Tech Stack
- Python
- Flask
- SQLite
- Win32 API

## ▶️ How to Run

### Start tracker
python tracker.py

### Start dashboard
python app.py

Then open:
http://localhost:5000
